# JavaScript Basics - Starter Code

Welcome to your first JavaScript programming mission!

## Getting Started

1. Fork this repository on GitHub
2. Clone to your computer
3. Open in VS Code
4. Open the integrated terminal (View â†’ Terminal or Ctrl+`)

## Files in This Project

- `hello.js` - Your first JavaScript file (already complete)
- `story.js` - Your dynamic story template (you'll build this)

## Running Your Code

In the terminal, type:

```bash
node hello.js
```

You should see a message! This means Node.js is working.

## Your Mission

Open `story.js` and follow the TODO comments to create your dynamic story.

Run your story anytime with:

```bash
node story.js
```
